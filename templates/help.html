Customer Lifetime Value Predictor.

How To Use?

  1. Enter the number of upcoming months that you want the valuation to cover.
  2. Select a CSV file (drag and drop into the box) or click on the drag and drop box to choose a file from your device.
  3. Wait for the file to be processed and the results to be displayed.

Note: The file MUST be in .csv format and follow the format provided on the right.

Frequently Asked Question (FAQ)

Q: What data should my CSV file contain?
A: The CSV file should contain transactional data. Do ensure that it matches the format displayed below

Q: What are the multiple files to download?
A: Each file represents different intervals of the CLV prediction, each with their distinct timestamp

Q: What is the diagram on the download page?
A: It is a bar graph that helps users visualize the segmentation of customers based on their CLV.

Q: Can I upload files other than CSV?
A: No, only CSV files are accepted. Ensure that they are formatted as specified, if not, the model will not work
